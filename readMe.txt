Compilation : Un makefile est fourni dans le dossier, il suffit d'executer la commande make

Exécution :

Il faut ensuite éxécuter la commande :

	./compilateur test.txt dest.txt

Avec :  test.txt : le fichier de code source
		dest.txt : le fichier de destination : s'il existe déjà un fichier avec ce nom, il sera écrasé
